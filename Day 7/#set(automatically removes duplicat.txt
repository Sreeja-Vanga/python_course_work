#set:(automatically removes duplicates)
->unordered
->mutable
->unique elements
->set allows only immutable data types in it
->we have no operations like repetition, concat,slicing, indexing,accessing in set except membership
#set operations:
->union(|) :to combine two sets
->intersection(&):to find common elements.
->difference(-):to remove elements in 1st set from 2nd set
->symmetric diff(^):to get all the elements.
->subset(<):to check whether all the elements of a set are present in the set
->disjoint():to get the elements which are not common
